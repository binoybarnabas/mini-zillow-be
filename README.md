# 🏠 Mini-Zillow Backend

This is the backend server for the **Mini-Zillow** project, built with **Node.js**, **Express**, and a relational DB (likely PostgreSQL or MySQL via Prisma).

---

## 🚀 Features

- 🔒 JWT Authentication
- 📄 RESTful APIs for properties, users
- 🌍 CORS configuration for frontend integration
- 🗂 Modular routing and middleware
- 🛠 Environment-based configuration
- 📦 Versel deployed

---

## 🛠 Tech Stack

- Node.js + Express
- Prisma ORM
- JWT for auth
- CORS middleware
- Dotenv for environment configs

---

## To run locally

- npm run dev

## For db migrations

- npm run migrate
